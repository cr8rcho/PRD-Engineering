# [+파일명] prd 파일 기준으로 화면 코드 업데이트 (prd -> edit code)

## 기능 파일: $ARGUMENTS

$ARGUMENTS 는 화면 페이지 코드 명(.tsx) 및 prd 페이지 명(.md)

prd 페이지(.md)와 화면 페이지 코드(.tsx)를 비교하고, 지금 prd 페이지(.md) 기준으로 화면 페이지 코드 (.tsx) 업데이트 반영.

prd 파일 위치 : prd_screens/
app 파일 위치 : src/

## 출력

prd 페이지(.md) 기준으로 화면 페이지 (.tsx) 코드 수정.
연결된 다른 파일의 수정이 필요하다면, 사용자에게 확인을 하고 진행.
